--exclude=<pattern> Skip untracked files matching pattern.

--exclude-from=<file> Read exclude patterns from <file>; 1 per line.

--exclude-per-directory=<file> Read additional exclude patterns that apply only to the directory and its subdirectories in <file>.

formatting:

<pathspec>… Files to show. If no files are given all files which match the other specified criteria are shown.
