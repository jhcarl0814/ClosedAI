Clone a repository into a new directory.

<repository> The (possibly remote) <repository> to clone from.

--recurse-submodules[=<pathspec>] After the clone is created, initialize and clone submodules within based on the provided <pathspec>. If no =<pathspec> is provided, all submodules are initialized and cloned. Is ignored if the cloned repository does not have a worktree/checkout (i.e. if any of --no-checkout, --bare, or --mirror is given).
