git branch

https://git-scm.com/docs/git-branch

List, create, or delete branches.

Choose an action to perform:

filtering:

by namespace:

by reachability:

--contains[=<commit>] Only list branches which contain <commit> (HEAD if not specified).

--merged[=<commit>] Only list branches whose tips are reachable from <commit> (HEAD if not specified).

--no-contains[=<commit>] Only list branches which don’t contain <commit> (HEAD if not specified).

--no-merged[=<commit>] Only list branches whose tips are not reachable from <commit> (HEAD if not specified).

by namespace-1refname:

<pattern>… Used as shell wildcards to restrict the output to matching branches. If multiple patterns are given, a branch is shown if it matches any of the patterns.

sorting:

--sort=<key> Sort based on the key given. (Defaults to branch.sort.)

formatting:

when not --verbose/--verbose --verbose:

when --verbose/--verbose --verbose:

<branch-name> The name of the branch to create.

<new-branch> The new name for an existing branch.

<new-branch> The new name for an existing branch.

<branch-name>… The name of the branch to delete.
