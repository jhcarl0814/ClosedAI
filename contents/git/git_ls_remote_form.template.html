git ls-remote [<repository> [<pattern>…]]

https://git-scm.com/docs/git-ls-remote

List references in a remote repository.

Choose an action to perform:

output:

what refs to show:

(when /--tags/--branches --tags:)

--sort=<key> Sort based on the key given.

server options:

--server-option=<option> Transmit the given string to the server when communicating using protocol version 2. (Defaults to remote.<name>.serverOption.)

<pattern>… When unspecified, all references, after filtering done with --heads and --tags, are shown. When specified, only references matching one or more of the given patterns are displayed.

<repository> The "remote" repository.
