git <command> [<args>]

https://git-scm.com/docs/git

configuration:

-c <name>=<value> Pass a configuration parameter to the command.

--config-env=<name>=<env_var_name> Give configuration variable <name> a value, where <env_var_name> is the name of an environment variable from which to retrieve the value.

scope:

-C <path> Run as if git was started in <path> instead of the current working directory.
